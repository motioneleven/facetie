# Fac√©tie

Adds a Presenter for Sunspot::Facet with a quick helper facet_item for your views.

## Installation

Add this line to your application's Gemfile:

    gem 'facetie'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install facetie

## Usage

`<%= facet_item(self, facet, :c) %>`
